# NLP for Twitter Sentiment Analysis
Face Mask Detection web application based on computer vision and deep learning, created using OpenCV, Flask and Tensorflow/Keras

Created using these libraries/framework:
- Flask 1.1.2
- Tensorflow 2.3.1
- Keras
- OpenCV
